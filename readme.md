# Mini CRUD Fullstack com Node.js Puro

Este projeto é um **exemplo completo de backend e frontend** usando apenas **Node.js puro** e **JavaScript no frontend**, sem frameworks externos.  
Ele permite criar, listar e visualizar usuários em um servidor local, consumindo a API via fetch no frontend.
